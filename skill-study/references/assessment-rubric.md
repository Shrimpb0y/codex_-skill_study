# Assessment Rubric

## Grading Priorities

Grade answers in this order:

1. Conceptual correctness.
2. Coverage of required points.
3. Correct technical terminology.
4. Example quality.
5. Exam-ready English expression.
6. Grammar and clarity.

## Feedback Pattern

Use this pattern for most answer reviews:

```text
总体评价
逐题批改
更标准答案
易错点
建议背诵表达
评分
```
Keep the tone supportive and concrete. Explain why an answer is usable or incomplete instead of only saying “correct” or “wrong”.

## Common Fixes

- If the learner gives a category instead of an example, request or provide a concrete example.
- If two concepts are mixed together, separate them with a short comparison table.
- If informal English expresses the right idea, preserve the meaning and rewrite it in exam-ready form.
- If the learner copies a model answer, explain its logic so it can be adapted to case-study questions.
- Accept Chinese answers for conceptual understanding, then provide an English exam version.

## Scoring Guide

- `9-10/10`: Accurate, complete, and exam-ready.
- `7-8/10`: Mostly correct; terminology or coverage needs improvement.
- `5-6/10`: Partial understanding with important points or examples missing.
- `<5/10`: Conceptual confusion; reteach before continuing.

## Exam Answer Upgrade

For explanation questions, use:

```text
Definition:
Reason:
Example:
Effect or limitation:
```

For comparison questions, use:

```text
Concept A:
Concept B:
Main difference:
Example:
```
